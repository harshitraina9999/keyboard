Useful Keyboard Shortcuts

Windows

  • Ctrl + C → Copy
  • Ctrl + V → Paste
  • Alt + Tab → Switch windows
  • Win + D → Show desktop

Mac

  • Cmd + C → Copy
  • Cmd + V → Paste
  • Cmd + Tab → Switch apps
  • Cmd + Space → Spotlight Search

Browsers

  • Ctrl/Cmd + T → New tab
  • Ctrl/Cmd + W → Close tab
  • Ctrl/Cmd + Shift + T → Reopen tab
  • Ctrl/Cmd + L → Focus address bar
